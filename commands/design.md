# /design - Design Phase

## Flow

1. @analyst researches architecture options
2. @reviewer evaluates approach
3. (Human decision if strategic)
4. @implementer creates interfaces/DTOs
5. @reviewer validates design
6. Iterate until APPROVED (max 3)
7. Auto-proceed to /implement

## Completion

- [ ] Architecture decided
- [ ] Interfaces/DTOs created
- [ ] @reviewer: APPROVED

## Human Intervention

Only for strategic architecture decisions.

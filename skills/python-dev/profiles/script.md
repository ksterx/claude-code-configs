# Script Profile

## Markers

- Flat `.py` files
- `requirements.txt`
- No `src/`

## Principles

- Minimal abstraction
- Self-contained
- Clear, readable code
- Quick execution

## Structure

```
./
├── main.py          # The script
├── requirements.txt # Dependencies
└── README.md        # Usage instructions
```

## Rules

- Keep in one file if < 300 lines
- Avoid unnecessary classes
- Use functions, not methods
- Prioritize readability over DRY
